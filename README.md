# gantt

* 本甘特图组件为JQ一把梭的结果
* data为数据文件，<b>timeData</b>为渲染表格列的数据，<b>lineContent</b>为渲染表格行，弹出面板，以及任务进度条的数据
* 没[线上地址](http://ernongxizhu.gitee.io/gantt/)是骗不来Star的，你点下试试
